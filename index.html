Problem

During high pressure periods inter hospital transfer decisions may be delayed or inconsistent due to limited visibility of hospital capacity and specialty availability.

Proposed Solution

This prototype demonstrates an AI assisted workflow that analyzes medical report text to infer severity and required specialty then ranks hospitals using capacity load distance and specialty match.

Prototype Features

  • Synthetic hospitals dataset
  • Synthetic medical transfer cases
  • NLP based specialty and severity prediction
  • Explainable hospital ranking engine
  • Interactive Streamlit demo

Data Governance

The current version uses synthetic data only. Real operational data can be integrated after administrative approval under proper privacy controls.

System Workflow

1

Input

Transfer request and medical report text

2

NLP Analysis

Infer severity level and required specialty

3

Hospital Ranking

Score hospitals based on capacity load distance and match

4

Recommendation

Return ranked hospitals with clear reasoning
